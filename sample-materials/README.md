# サンプル素材（名刺・領収書）

AI研修のデモやハンズオンで使う、スキャン練習用の画像素材です。

| フォルダ | 中身 | 枚数 |
|---|---|---|
| [`cards/`](cards/) | 名刺の写真 | 20枚 |
| [`receipts/`](receipts/) | 領収書の写真 | 20枚 |

すべて**架空の会社・人物・取引**です。実在の個人情報は含まれていません（電話番号は `03-1234-5678` のようなダミー、メールも実在しないドメイン）。

## 使いどころ

研修では、この素材を使って次のような演習ができます。

### 名刺（`cards/`）
- 20枚をスキャンして、会社名・部署・役職・氏名・住所・TEL・メールを一覧化する
- 会社名や役職から事業内容を推測し、営業メールの下書きを作る

### 領収書（`receipts/`）
- 日付・宛名・金額・但し書き・発行元・インボイス登録番号を読み取って経費精算表にする
- 勘定科目に仕分ける／登録番号の有無をチェックする

名刺と領収書は取引先名で突き合わせできます（例: `cards/card_12.jpg` の株式会社テクノブリッジ と `receipts/receipt_01.jpg` の宛名が一致）。
